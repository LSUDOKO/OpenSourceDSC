JavaScript Unit Tests

This page runs unit tests for JavaScript functions in the project. Click the "Run Tests" button to execute all tests.

Tests are organized in test suites for different components:

  • Gallery - Tests for gallery image display and filtering
  • Portfolio - Tests for project rendering and contact form
  • Testimonials - Tests for testimonial display and carousel functionality
  • Theme - Tests for theme toggling and persistence

These tests utilize a simple custom testing framework without third-party dependencies.

The framework provides these assertion methods:

  • assert.equal(actual, expected, message) - Tests that two values are strictly equal
  • assert.true(value, message) - Tests that a value is true
  • assert.false(value, message) - Tests that a value is false
